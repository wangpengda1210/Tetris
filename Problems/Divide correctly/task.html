Divide correctly

Adele is just learning to program, and she wrote a function that would divide 100 by the given number with certain conditions:

def divide_by(number):
    if number > 0:
        print(100 / number)
    else:
        print(100 / (number / 2))

    print(100 / (number / 2))
ZeroDivisionError: division by zero
