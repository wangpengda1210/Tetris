Trace the inversion

In the topic, we mentioned that SciPy and NumPy have some functions in common. The goal of this problem is to illustrate that in some cases those libraries are interchangeable.
Given a matrix \(A = \begin{pmatrix} 1 & 2\\ 2 & 3 \end{pmatrix}\), use SciPy or NumPy to find the trace of the inverse of the matrix \(A\). Print out your answer.

Use numpy.trace or scipy.trace to get the trace of a matrix. To find the inverse, use inv function from the linalg subpackage.
